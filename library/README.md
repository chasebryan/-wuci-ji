# WUCI-JI Library

This directory is the durable library for human-readable documentation indexes
and report results produced by local verification cycles.

- `documentation/` is for curated pointers to stable project documentation.
- `reports/` is for dated reports from test cycles, evidence reviews, and
  defensive assessment runs.

Library entries are evidence catalogs and run reports. They are not production
authority, external review, runtime-containment evidence, quantum-safety
evidence, or score changes unless a referenced proof lane explicitly says so.
