# Daylight Specs

This directory holds the active Daylight specification and hardening notes.

- `daylight-minimal-core-v0.4.md` is the current minimal math core target.
- `daylight-envelope-hardening-v0.5.1-2.md` records the implemented v6
  byte-level hardening layer.
- `daylight-envelope-math-spec-v0.3.md` is the prior envelope math baseline.

Imported source references are kept in `../references/` instead of mixed into
the active spec set.
