# Daylight References

This directory holds imported or upstream source material used as Daylight
references. These files are kept for provenance and comparison, not as the
current implementation target.

- `dlv0.5/2.md` is the imported v0.5 minimal math core reference.
- `dlv0.5/v0.6M1-HARDENING.md` is the imported v0.6 M1 hardening reference.

Active working specs live in `../specs/`.
