# Yocto Build Room

Execution class: `heavy-source-build-system-room`

This directory defines the controlled measuring chamber for the Yocto candidate. It does not run BitBake and does not clone source trees.

Blocked until local Yocto/Poky source or an approved source acquisition policy exists, a controlled output path policy exists, a controlled backend is usable, and a future explicit build attempt authorization exists.
