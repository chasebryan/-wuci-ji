# Direct Rootfs Buildroom Preparation

Phase 3C-B records direct-rootfs preparation policy for Buildroot, Alpine, Debian minimal, and Void.

This directory contains policy only. The files here do not authorize rootfs generation, substrate artifact attempts, runtime inspection, substrate selection, candidate ranking, or numeric WuciOS scoring.

Generated Phase 3C-B reports and optional non-artifact scaffolding are written under `build/wucios/`.
