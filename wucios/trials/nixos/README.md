# NixOS Euclid Phase 2 Probe

This directory contains the tracked NixOS build feasibility probe for WuciOS v2.4 Euclid Trial Phase 2B.

Default mode is `SAFE_DETECT_ONLY`. It detects local Nix tooling and records host-store blockers without writing to `/nix/store`.

NixOS remains unranked. WuciOS does not select a substrate in Phase 2B.
