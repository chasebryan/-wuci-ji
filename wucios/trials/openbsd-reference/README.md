# OpenBSD Reference Euclid Phase 2 Probe

This directory contains the tracked OpenBSD reference build feasibility probe for WuciOS v2.4 Euclid Trial Phase 2B.

Default mode is `SAFE_DETECT_ONLY`. It detects local reference images and runtime tooling without downloading images, installing systems, or launching a VM.

OpenBSD reference is a non-Linux reference path. It remains unranked and does not select a WuciOS substrate.
