# GNU Guix Euclid Phase 2 Probe

This directory contains the tracked GNU Guix build feasibility probe for WuciOS v2.4 Euclid Trial Phase 2B.

Default mode is `SAFE_DETECT_ONLY`. It detects local Guix tooling and records host-store or daemon-policy blockers without starting daemons.

Guix remains unranked. WuciOS does not select a substrate in Phase 2B.
