# Yocto Euclid Phase 2 Probe

This directory contains the tracked Yocto build feasibility probe for WuciOS v2.4 Euclid Trial Phase 2B.

Default mode is `SAFE_DETECT_ONLY`. It detects BitBake, host tooling, and a local Yocto/Poky source tree without cloning sources.

Yocto remains unranked. WuciOS does not select a substrate in Phase 2B.
