DAYLIGHT ENVELOPE — MINIMAL MATH CORE v0.6
M1-HARDENING DELTA OVER v0.5

Status(Daylight_v0.6) = research_draft
MaturityClaim(Daylight_v0.6) = M0+
M1_Target(Daylight_v0.6) = 1

ProductionAllowed(Daylight) = 0 until Maturity(Daylight) >= M6

v0.6 is not a new primitive and not a production protocol.
v0.6 is a byte-level hardening pass over v0.5.


0. V0.6 PURPOSE

v0.6 targets the remaining v0.5 deductions:

  1. Reduce implementation ambiguity.
  2. Remove hash/transcript ambiguity.
  3. Freeze mandatory object schemas.
  4. Replace abstract PolicyEval with a minimal deterministic policy object.
  5. Specify AuthBlock byte structure.
  6. Specify exact PublicPreOK order.
  7. Separate mandatory core from extensions more aggressively.
  8. Define conformance levels and rejection stages.
  9. Make test-vector generation directly implementable.

v0.6 does not claim:
  external review,
  formal proof completion,
  reference implementation completion,
  production readiness,
  or global 256-bit security.

3. DIGEST CONVENTION FIX

v0.5 used H(T0) where T0 is already encoded bytes.
v0.6 removes this ambiguity.

Canonical object hash:

  HC(x) = SHA3-512(ENC(x))

Raw byte hash:

  HB(b) = SHA3-512(b)

Short canonical hash:

  HC32(x) = SHAKE256(ENC(x),32)

Short byte hash:

  HB32(b) = SHAKE256(b,32)

Transcript bytes:

  T0 = E("daylight.pre.v6", header)
  h0 = HB(T0)

  T1 = E("daylight.auth.transcript.v6", ...)
  h1 = HB(T1)

Rule:

  If input is a typed object, use HC.
  If input is already encoded transcript bytes, use HB.

Forbidden:

  H(T0)
  H(T1)
  H(kem_block)

Required replacements:

  H(T0)       -> h0 = HB(T0)
  H(T1)       -> h1 = HB(T1)
  H(kem_block)-> HC(kem_block)


4. HASH AND KDF

HC(x) = SHA3-512(ENC(x))
HB(b) = SHA3-512(b)

ArtifactHash(A) = HB(A)

KDF.Extract(salt,ikm) = HKDF-SHA512.Extract(salt,ikm)
KDF.Expand(prk,info,L) = HKDF-SHA512.Expand(prk,info,L)

KDF2(secret,label,input,L) =
  KDF.Expand(
    KDF.Extract(
      HC(["daylight-kdf2-salt.v6", label]),
      secret
    ),
    ENC(["daylight-kdf2-info.v6", label, input]),
    L
  )

All KDF labels are ASCII and versioned.

18. AUTHORIZATION

Q_m = {
  q in auth_block.q_sigs :
    q.key_id identifies pk_Q in Q_roster
    and ML-DSA-87.Verify(pk_Q, AuthMsg, q.sig, ctx_auth) = 1
    and KeyLive(pk_Q, prev_log_head) = 1

V_Q =
  1[cardinality(Q_m) >= t_Q]
  * 1[domain_count(Q_m) >= u_Q]
  * UniqueKeys(Q_m)

V_H =
  if H required:
    1[auth_block.h_sig != null]
    * SLH-DSA-SHAKE-256s.Verify(pk_H, AuthMsg, h_sig, ctx_auth)
    * KeyLive(pk_H, prev_log_head)
  else:
    1[auth_block.h_sig = null]

V_F =
  if F required:
    FROST verification predicate
  else:
    1[auth_block.frost_auth = null]

V_Auth(profile,r,mu) =
  product over X in Req(profile,r,mu) of V_X

20. FIXED PUBLIC PRECHECK ORDER

PublicPreOK is not just a predicate.
It is an ordered procedure.

P0. Decode omega as deterministic CBOR.
P1. Check Envelope_v6 schema.
P2. Check Header_v6 schema.
P3. Check suite_id, version, profile, r, mu, action, content_scope, aead_id.
P4. Check AuxBlock_v6 schema and object hashes.
P5. Parse policy_obj, keyset_obj, and claims_obj.
P6. Check static policy gate.
P7. Check KEMBlock_v6 public shape and key references.
P8. Build T0, h0, T1, h1, AuthMsg.
P9. Check AuthBlock_v6 schema.
P10. Verify authorization signatures and quorum predicates.
P11. Check content review preconditions.
P12. Check no-downgrade predicates.
P13. Check log proof if required.
P14. Check install and witness predicates if required.

If any stage P0..P14 fails:

  Open(omega) = bottom
  no private KEM decapsulation is called
  AEAD.Dec is not called
  plaintext is not materialized

PublicPreOK = 1 iff P0..P14 all pass.


21. PRIVATE OPEN ORDER

After PublicPreOK = 1:

S0. Decapsulate ML-KEM.
S1. Decapsulate DHKEM.
S2. Derive K_E, K_COM, N0.
S3. AEAD.Dec(K_E, N0, C, AD=T0).
S4. Decode PrivatePayload_v6.
S5. Check artifact commitment.
S6. Check leak/review private consistency.
S7. Zeroize secrets and return artifact.

If S0..S6 fails:

  Open = bottom
  all derived secrets are zeroized
  plaintext buffers are zeroized
