# Wuci-Ji External Validation Uplift

External validation can strengthen authority when it is scoped, signed, pinned,
non-fixture, reproducible where practical, and tied to the reviewed artifacts.

## What External Validation Changes

External validation can elevate:

- Reviewer trust.
- Conformance level.
- Audit authority.
- Score confidence.
- Customer adoption.
- Procurement readiness.

## What External Validation Does Not Change

External validation does not automatically prove:

- Universal security.
- No vulnerabilities.
- Permanent safety.
- Government approval unless issued by the actual authority.
- Certification unless performed under the actual certification scheme.
- Runtime containment outside the reviewed runtime boundary.
- Quantum safety from classical-only evidence.

## Accepted Evidence Types

- Signed independent rebuild receipts.
- Third-party security review reports.
- External firewall-profile reviews.
- Red-team findings and remediation records.
- Falsification results.
- Certification or authorization records from the actual issuing body.

## Attestation Requirements

An external attestation must include:

- Attestor identity.
- Attestation type.
- Scope.
- Reviewed commit.
- Reviewed artifacts.
- Signature.
- Signature scheme.
- Public key reference.
- Non-claim acknowledgement.
- `claim_usable: true` only when the evidence is non-fixture and scoped.

## D9 Rule

D9 formal authority requires actual external certification, authorization, or
regulatory authority. Wuci-Ji can carry evidence for D9, but cannot self-issue
D9.
