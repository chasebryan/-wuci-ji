# Wuci-Ji Security Product MVP

Product name suggestion:

```text
Wuci-Ji Daylight Standard
```

The MVP is an evidence-bound security assurance product, not an endpoint
protection system.

## MVP Capabilities

1. Claim scanner.
2. Evidence registry.
3. Deterministic scorecard generator.
4. Release gate.
5. SBOM/provenance binder.
6. Conformance report.
7. Control-map export.
8. Audit packet generator.
9. Website/status JSON verifier.
10. CI/CD action.

## Not in MVP

- Endpoint protection.
- Malware detection.
- Network firewalling.
- Runtime sandboxing.
- Kernel enforcement.
- Production cryptography.
- FIPS validation.
- Automatic vulnerability fixing.

## MVP Customers

- Open-source maintainers.
- Security-conscious companies.
- Internal platform teams.
- Release engineering teams.
- Auditors.
- Research labs.
- Defense-adjacent software teams needing proof discipline.

## MVP Value

The MVP prevents unsupported security claims and unsafe releases by requiring
evidence-bound proof before authority.

## Release Target

A v0 release should provide:

- Versioned schemas.
- Example evidence.
- CLI validation.
- CI workflow.
- Product boundary docs.
- Claim-evidence map.
- Vulnerability response model.
- Non-claim enforcement.
