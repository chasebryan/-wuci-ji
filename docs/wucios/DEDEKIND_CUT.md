# Dedekind Cut

Dedekind Cut separates Noether Core, Birkhoff Bastion, Tarski Review Appliance, and Developer Desktop.

Noether Core is the serious release profile. Birkhoff Bastion is optional operator shell material. Tarski Review Appliance produces review evidence. Developer Desktop is non-authoritative convenience material.

Developer Desktop may contain desktop environments, browsers, editors, compilers, debuggers, and build tooling. It must never be confused with Noether Core and must not be used for release-authoritative scoring unless explicitly marked diagnostic.
