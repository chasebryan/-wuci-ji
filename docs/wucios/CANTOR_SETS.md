# Cantor Sets

Cantor Sets are explicit allowlists and denylists for packages, services, claims, and capabilities.

Current set files:

- [cantor-allowed-noether-packages.txt](../../wucios/sets/cantor-allowed-noether-packages.txt)
- [cantor-denied-noether-packages.txt](../../wucios/sets/cantor-denied-noether-packages.txt)
- [cantor-denied-default-services.txt](../../wucios/sets/cantor-denied-default-services.txt)
- [cantor-denied-claim-phrases.txt](../../wucios/sets/cantor-denied-claim-phrases.txt)
- [cantor-allowed-review-outputs.txt](../../wucios/sets/cantor-allowed-review-outputs.txt)

Any exception must be explicit and reviewable. Exceptions must not silently hide a current claim violation.
