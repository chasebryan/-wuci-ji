# Shannon Ledger

Shannon Ledger tracks information exposure in WuciOS v2.4.

It covers:

- network ports
- enabled services
- logs
- secrets
- keys
- telemetry
- artifact leakage
- entropy-sensitive materials

No secrets may be placed in public artifacts. Public review packets must contain evidence, manifests, hashes, non-claims, and measurements only.

If exposure cannot be measured from a current artifact, record `NOT_MEASURED`.
