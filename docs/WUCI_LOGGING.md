# WUCI Logging Evidence

This file documents the local logging boundary for a fresh Wuci-Ji checkout.
Operational logs and command transcripts should be written under `build/logs/`
or another private operator-selected path that is outside public evidence.

Logs must not contain secrets, tokens, private keys, host-specific identities,
or unredacted absolute local paths when copied into public reports.

This document is configuration evidence only. It does not prove log retention,
centralized collection, external auditability, production readiness, or host
cleanliness.
