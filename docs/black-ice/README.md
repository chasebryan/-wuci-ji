# WUCI-BLACK-ICE

`WUCI-BLACK-ICE` is now the compatibility name for
[`WUCI-NOXFRAME`](../noxframe/README.md).

The old command still works:

```sh
tools/wuci-black-ice --profile full
make black-ice-launch
```

Without `--profile full`, the compatibility command uses the same 7-day
NOXFRAME auto clock as `tools/wuci-noxframe`.

New docs, reports, and the self-seal live under:

```text
docs/noxframe/
```
