Product Boundary

The public evidence surface shows claim discipline. It does not claim more than its evidence supports.

Allowed Claims

Research/proof artifact, evidence-bound release and claim-integrity surface, and public-review handles.

Forbidden Claims

Production cryptography, general runtime sandbox, post-quantum secure, independent audit, government approval, cATO/RMF/FIPS/FedRAMP/NIAP authority.

Current Security Boundary

Only implementation boundaries documented in the repository and backed by tests can be treated as enforced.

Product Category Boundary

Evidence-bound security assurance, not endpoint protection or security-operations replacement.

Runtime Boundary

General runtime containment is future-gated until OS-level enforcement, review, and proof lanes exist.

Cryptographic Boundary

Classical signatures and fixture authority do not create quantum-safe or production authority claims.

Certification Boundary

Control maps are evidence indexes. Certification requires the actual issuing scheme.

Non-Claim Warning

Wuci-Ji / Daylight is not currently a production security replacement. It can verify claims, gate releases, map controls, and expose unproven assumptions.
