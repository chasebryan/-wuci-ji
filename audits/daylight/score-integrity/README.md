# Daylight Score-Integrity Audits

This directory records Daylight score-integrity audit passes.

It contains claim ledgers, math, logic, recomputation records, public-surface
comparisons, and final reports. Each run is keyed to an audited commit and keeps
the generated JSON reports beside human-readable notes.

It does not certify security, production readiness, audit status,
post-quantum security, agency endorsement, or mathematical finality. It does
not declare Singularity.
