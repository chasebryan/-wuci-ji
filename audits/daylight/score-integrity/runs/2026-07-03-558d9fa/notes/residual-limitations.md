# Residual Limitations

The run preserves these limitations:

- Historical v13 percentage wording predates a documented rounding rule; the
  generated ratio report records the rule that matches the stated values.
- DaylightNPT file and number counts are properties of the scanned tree. Each
  run must record the exact values it observes.
- Claims about score values outside the repository remain out of scope unless
  the repository quotes and bounds them.
- The score-integrity tool checks the enumerated distinct claims. DaylightNPT
  remains the line-level gate for new numeric claims.
